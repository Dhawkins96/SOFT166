About This Project

This project was to design a Password Strength Checker and integrate them with the Hue lights in SMB 109. If you haven't already looked at the Project the link is in the side bar to the left. Within the Password Strength Checker it changes automatically when you type anything in the password text-box. This then scores you to see if you have a strong password based on what you have typed so if you have a capital letter, a number, a special character and the length. This score gets put into a progress bar and goes up as you type in the relevant information and a set of checkboxes are listed underneath showing what you need to get the strongest passwords. While all of this is happening the Hue lights are also changing depending on the strength of the password, if you are ever unsure about what a colour means you can click on the Light Key Index in the Password Strength Checker which explains all the colours and how strong the password is.
I hope you enjoy!
